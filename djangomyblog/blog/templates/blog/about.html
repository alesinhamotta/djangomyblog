Sobre o Projeto

Este projeto foi desenvolvido como parte do Desafio II do curso de Django, com o objetivo de consolidar conceitos fundamentais de desenvolvimento web utilizando o framework Django.

A aplicação consiste em um blog completo, onde usuários podem criar uma conta, realizar login, publicar posts e interagir através de comentários. Todo o sistema foi construído seguindo boas práticas de organização, segurança e reutilização de código.

Entre os principais recursos do projeto estão:

  • Cadastro e autenticação de usuários
  • CRUD completo de posts
  • Sistema de comentários vinculados aos posts
  • Separação de estilos CSS para melhor organização
  • Interface simples, responsiva e funcional

O projeto também foi preparado para implantação em ambiente de produção, demonstrando o fluxo completo desde o desenvolvimento local até o deploy.
